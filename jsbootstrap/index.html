Welcome to RdbHost!

These linked pages are a demonstration of programming for Rdbhost.com. They include a simple DVD tracking application, an app to send email, and a demo of handling OpenID logins for your users.

DVD DB Demo

These two pages represent different roles.

With the first page, you can use the Super role to setup the database, schema, and tables.

With the second, you can train the server account; once the training is done, the second can be used safely by anyone to manipulate the database, as only white-listed queries can be executed on that role.

Config » Preauth »

Emailing Demo

These two pages represent different roles.

With the first page, you can use the Super role to setup the keyfile.

With the second, you can train the server account; once the training is done, the second can be used safely by anyone to send emails.

Config » Preauth »

OpenID Demo

These two pages represent different roles.

The first page is non-essential, as the necessary server configuration is done for you when you enable OpenID. This is here as a guide, should you wish to use a different post-login response.

With the second, you can test logging in using an OpenID login. This example uses the reader role, but any role can be used for OpenID logins.
